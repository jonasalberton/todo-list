# Todo list

Click in this link to access the live app hosted on Netlify https://todoalberton.netlify.app/

This is a simple app using:
- React
- Typescript
- Stitches.css
- Context API
- Redux
- Jest

## Getting started
```
git pull https://github.com/jonasalberton/todo-list.git
cd todo-list
npm i
npm run dev
```
